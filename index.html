    Introduction

    Welcome to the Tport API! You can use our API to access Tport API endpoints.

    Authentication

    SHA256 signature:

    echo -n '{"timestamp":1499827319559}' | openssl dgst -sha256 -hmac "EUqBu7ECBxzciFDYDNHuDHtkka2KlShVvzw9mJ3iJUJIrGCC-WQVN2V1FsvuS7yR"

    The above command returns:

    (stdin)= 58ff490d12511aad632ce2dad6583ad7e9347378ea401ef7c017ab2058fb5404

    Tport uses API keys to allow access to the API. You can register a new Tport API key at our developer portal.

    Here is a step-by-step example of how to send a vaild signed payload from the Linux command line using echo, openssl, and curl.

    Key Value
    apiKey 5E8SVNzr8sPqaxoP9ZN9xjeV6PwRKCOScEhyG7CdKzzuDjARFnT7RnV6mRIjvEvl
    secretKey EUqBu7ECBxzciFDYDNHuDHtkka2KlShVvzw9mJ3iJUJIrGCC-WQVN2V1FsvuS7yR
    You must replace apiKey and secretKey with your personal API keys.

-H "Signature: 58ff490d12511aad632ce2dad6583ad7e9347378ea401ef7c017ab2058fb5404" \
-X GET \
--data '{"timestamp":1499827319559}' \
https://api.tport.nl/rest/v1/balances

    Withdrawal

    curl --header "Content-Type: application/json" \
-H "X-TPORT-APIKEY: 5E8SVNzr8sPqaxoP9ZN9xjeV6PwRKCOScEhyG7CdKzzuDjARFnT7RnV6mRIjvEvl" \
-H "Signature: 9a83204e6143037f2c61b03ef880624cd24dfc52346daaaf0e60a02592b0547e" \
-X POST \
--data '{"wallet":"BTC-40277ZW","amount":0.001,"system":1,"to_currency":"BTC","address":"1234","timestamp":1499827319559}' \
https://api.tport.nl/rest/v1/withdrawal

    $api = new TportApi();
$data = [
    'wallet' => 'BTC-40277ZW',
    'amount' => 0.001,
    'system' => 1,
    'to_currency' => 'BTC',
    'address' => '1234',
];
$api->query('withdrawal', $data, 'POST', true);

    The above command returns JSON structured like this:

    {
  "success": 1,
  "data": {
    "id": 368,
    "status": "Pending"
  }
}

    This endpoint withdrawal.

    HTTP Request

    POST https://api.tport.nl/rest/v1/withdrawal

    Query Parameters

    Parameter Mandatory Description
    wallet Yes Wallet is a place from which removes an amount of money
    amount Yes Quantity of money
    system Yes System ID @see Get Payment Systems
    to_currency Yes Which currency you want to receive
    address Yes Address to receive money

    Errors

    The Tport API uses the following error codes:

    Errors

    Error Code Meaning
    105 Parameters validation error
    106 API key is wrong
    107 Signature is wrong
    108 Timestamp is wrong
    109 Timeout error
    110 Signature validation error

    System Errors

    Error Code Meaning
    400 Bad Request
    403 Forbidden
    404 Not Found
    500 Internal Server Error -- We had a problem with our server. Try again later.
